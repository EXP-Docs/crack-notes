---FASM PDK for Decompiler by GPcH---

Содержимое:
readme.txt - описание
DecompilerPlugin.asm   - исходный файл
DecompilerPlugin.dll   - очевидно скомпилированный файл :)

Данный исходный код служит примером того, как использовать замечательный транслятор ассемблера "Flat Assembler"
для написания плагинов для Decompiler'a.

Исходный код распространяется абсолютно бесплатно и без каких-либо ограничений (fasm, кстате тоже :))

Для сборки плагина нужен fasm любой из доступных Вам версий (на момент создания этого плагина была использована версия
1.67.23). Единственная проблема, которая может возникнуть при компилировании - это сообщение компилятора об ошибке,
что подключаемый файл в директиве include не найден. В этом случае исправьте строку в соответствии с местоположением
подключаемых файлов на Вашем компьютере (как правило это папка Include в директории самого fasm'a).
После компилирования в папке с исходным файлом появится скомпилированный файл с расширением .dll. Для того, чтобы
Decompiler смог запускать этот плагин, его нужно поместить в папку "plugins" самого декомпилятора. В результате в меню
"Plugins" в программе появится пункт "FASM plugin".

О самом плагине:
При запуске плагина на экран будет выведено окно, состоящее из строки с указанием полного пути к декомпилируемому файлу,
а также текстовое поле и четыре кнопки - Save file, Get text, About и Exit. При нажатии первой кнопки будет сохранено содержимое
текстового поля в файл с именем "decompiled.rtf", при нажатии на вторую в текстовое поле будет скопировано
содержимое активного окна, при нажатии третьей - будет выведена информация о разработчике, при нажатии четвёртой  плагин завершит
свою работу.
